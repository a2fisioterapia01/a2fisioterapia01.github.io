# Imágenes de Servicios

Esta carpeta contiene las imágenes para las cards de servicios en la sección de servicios.

## Imágenes requeridas:

1. **tratamientos-personalizados.jpg** - Imagen para tratamientos personalizados
2. **pilates.jpg** - Imagen para Pilates Máquinas + Suelo
3. **tecnologia.jpg** - Imagen para Tecnología Avanzada (ecografía, EPI, etc.)
4. **lesiones-deportivas.jpg** - Imagen para Lesiones Deportivas
5. **rehabilitacion.jpg** - Imagen para Rehabilitación Postquirúrgica
6. **domicilio.jpg** - Imagen para Fisioterapia a Domicilio

## Especificaciones:

- **Formato recomendado**: JPG o WebP
- **Dimensiones**: Mínimo 800x600px (relación de aspecto 4:3 o similar)
- **Peso**: Optimizado para web (< 200KB por imagen)
- **Estilo**: Imágenes profesionales relacionadas con cada servicio

## Notas:

Las imágenes se mostrarán con:
- Altura fija de 192px (h-48 en Tailwind)
- Efecto hover de zoom suave (scale-105)
- Esquinas redondeadas superiores
- Ícono circular superpuesto en la parte inferior central
